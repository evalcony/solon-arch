# solon-arch
基于 solon 框架开发的 web 项目脚手架

[solon 文档](https://solon.noear.org/article/learn-start)

已支持

- xxl-job
- redis
- redission
- mybatis-plus
- HikariCP
- caffeine (高性能本地缓存)
- [hutool, 文档在此](https://www.hutool.cn/docs/#/) 
  - [API 在此](https://apidoc.gitee.com/dromara/hutool/)
- [fastjson](https://github.com/alibaba/fastjson)
